# Cotizador Seguro

## Descripción
Proyecto cotizador seguro

## Tecnologías Utilizadas
- HTML5
- CSS3
- Sass
- JavaScript
- JSX
- React
- Node 14.5.4

## Comandos para ejecución del proyecto

Instalar todas las dependencias:
```
npm run init
```

Ejecutar servidor de desarrollo:
```
npm start
```
